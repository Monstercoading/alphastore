# Alpha Store Logo

## الشعار المخصص

الرجاء وضع شعار Alpha Store في هذا المجلد باسم:
- `alpha-store-logo.png`

## المواصفات المطلوبة:

- **الصيغة:** PNG بخلفية شفافة (transparent)
- **الحجم:** يفضل أن يكون بحجم 200x50 بكسل أو مشابه
- **الجودة:** عالية الجودة (300 DPI)
- **الخلفية:** شفافة للتناسق مع الستايل الداكن

## الاستخدام في الكود:

تم تحديث الكود التالي لاستخدام الشعار المخصص:

### Navbar:
```jsx
<img 
  src="/alpha-store-logo.png" 
  alt="Alpha Store" 
  className="h-10 w-auto transition-transform duration-300 group-hover:scale-105"
/>
```

### صفحات تسجيل الدخول وإنشاء الحساب:
```jsx
<img 
  src="/alpha-store-logo.png" 
  alt="Alpha Store" 
  className="h-16 w-auto transition-transform duration-300 group-hover:scale-105"
/>
```
