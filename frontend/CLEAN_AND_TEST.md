# خطوات تنظيف واختبار النظام

## 1. تنظيف localStorage
افحص المتصفح وافتح Console ثم نفذ:
```javascript
localStorage.clear();
```

## 2. اختبار تسجيل الدخول العادي
1. اذهب إلى `/register`
2. أنشئ حساب جديد (مثلاً: test@example.com)
3. يجب أن ينتقل لصفحة `/profile`
4. في صفحة الملف الشخصي، جرب تعديل الاسم فقط
5. الإيميل يجب أن يكون للعرض فقط

## 3. اختبار Google OAuth
1. اذهب إلى `/login`
2. انقر على زر Google
3. سينشئ مستخدم googleuser@gmail.com
4. يجب أن ينتقل لصفحة `/profile`
5. الإيميل يجب أن يكون للعرض فقط

## 4. التحقق من قائمة المستخدمين
في Console، تحقق:
```javascript
console.log(JSON.parse(localStorage.getItem('users') || '[]'));
```

## 5. اختبار تسجيل الخروج والدخول مرة أخرى
1. سجل خروج من صفحة الملف الشخصي
2. حاول تسجيل الدخول بنفس البيانات
3. يجب أن يعمل بشكل صحيح
