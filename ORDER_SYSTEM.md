# نظام إدارة الطلبات للإنتاج

## الوصف
نظام متكامل لإدارة الطلبات في بيئة الإنتاج مع دعم الاستيراد والتصدير.

## المميزات

### ✅ Backend API Endpoints
```javascript
// تصدير الطلبات
GET /api/orders/export

// استيراد الطلبات  
POST /api/orders/import

// إحصائيات الطلبات
GET /api/orders/stats

// تحديث حالة الطلب
PUT /api/orders/:id
```

### ✅ Frontend Functions
- **تصدير الطلبات**: تحميل ملف JSON من السيرفر
- **استيراد الطلبات**: رفع ملف JSON إلى السيرفر
- **مزامنة البيانات**: تحديث الواجهة تلقائياً

## طريقة العمل في الإنتاج

### 1. تصدير الطلبات
```javascript
// من لوحة التحكم
exportOrders() → API → Download JSON file
```

### 2. استيراد الطلبات
```javascript
// من لوحة التحكم  
importOrders(file) → API → Update database
```

### 3. مزامنة البيانات
```javascript
// تلقائي بعد كل عملية
localStorage ← API ← JSON files
```

## الفوائد للإستضافة

### 🔄 **مزامنة بين الأجهزة**
- الطلبات محفوظة في السيرفر
- يمكن الوصول من أي جهاز
- لا فقدان للبيانات

### 💾 **نسخ احتياطي تلقائي**
- ملفات JSON في مجلد `data/`
- يمكن استيرادها بسهولة
- حماية من فقدان البيانات

### 📊 **إحصائيات فورية**
- عدد الطلبات الكلي
- الطلبات حسب الحالة
- إجمالي الإيرادات

## ملفات الإنتاج

```
backend/
├── data/
│   ├── orders.json      # بيانات الطلبات
│   ├── products.json    # بيانات المنتجات
│   └── users.json       # بيانات المستخدمين
└── server.js          # API مع نظام الطلبات
```

## استخدام في الإستضافة

### 1. رفع الملفات
```bash
# رفع مجلد production كامل
production/
├── backend/
├── frontend/
└── start.sh
```

### 2. تشغيل السيرفر
```bash
cd backend
npm start
# السيرفر يعمل على المنفذ 5000
```

### 3. الوصول للوحة التحكم
```
http://your-domain.com/admin
```

## اختبار النظام

### اختبار التصدير
1. اذهب للوحة التحكم
2. اضغط "تصدير الطلبات"
3. يجب تحميل ملف JSON

### اختبار الاستيراد
1. اضغط "استيراد طلبات"
2. اختر ملف JSON صالح
3. يجب تحديث قائمة الطلبات

## استكشاف الأخطاء

### مشاكل شائعة
- **CORS Error**: تحقق من إعدادات النطاقات
- **File Upload**: تحقق من حجم الملفات
- **Permission Error**: تحقق من صلاحيات مجلد `data/`

### الحلول
```javascript
// للتحقق من الاتصال
fetch('/api/orders/stats')
  .then(res => res.json())
  .then(data => console.log('Stats:', data));
```
